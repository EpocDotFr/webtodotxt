# Web Todo.txt

A web-based GUI to manage a [Todo.txt](http://todotxt.com/) file.

## Prerequisites

  - Should work on any Python 3.x version
  - A WSGI-capable web server

## Installation

Clone this repo, and then the usual `pip install -r requirements.txt`.
